windows exporter添加认证一键部署脚本

2024-07-04
Scripts
字数统计: 657   |   阅读时长≈ 3 分钟

这个脚本主要完成了以下几个任务：

在指定目录创建Windows Exporter的安装目录。
通过FTP从服务器下载配置文件config.yml和Windows Exporter的安装包。
安装Windows Exporter MSI包，并配置启用的收集器和监听端口。
管理Windows Exporter服务：停止并删除可能存在的旧服务，创建新的服务定义，配置服务自动启动、故障恢复策略，并启动服务。
最后更新服务的描述信息，并在脚本执行结束时暂停，以便查看输出信息。
