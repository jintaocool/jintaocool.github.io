ElastiFlow介绍

ElastiFlow目前为开源的流量分析系统，可接收netflow、sflow和ipfix，同类商业软件参考solarwinds NTA

前置条件：安装docker和docker-compose

docker：（yum -y install docker-ce）

docker-compose：（pip3 install docker-compose -i http://mirrors.aliyun.com/pypi/simple/ –trusted-host http://pypi.aliyun.com）

准备时间校对文件

未配置可能导致Discover界面中的timestamp参数慢8小时

准备docker-compose.yml文件

使用docker-compose.yml部署，注意此文件需要单独放在业务文件盘符中，放置/目录存放容器数据过大导致无法存储netflow数据。如：新建/data文件夹，mount挂载新盘sdb
